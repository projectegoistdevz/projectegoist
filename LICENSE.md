## 🔧 Troubleshooting & Common Issues

### ❌ Script Not Injecting

**Symptoms:** Nothing happens after hitting Execute; no panel appears.

**Solutions:**
- Ensure the game is fully loaded before injecting *(wait for the main lobby screen)*
- Restart your executor and re-attach
- Check that you are using the latest script version from the official download page
- Temporarily disable antivirus — it may block the injection DLL

---

### 🐢 Admin Panel Lagging or Freezing

**Symptoms:** The floating panel stutters or becomes unresponsive.

**Solutions:**
- Lower `SPEED_MULTIPLIER` to `1.0` to reduce CPU load
- Disable particle effects using the visual settings tab
- Close background apps, especially on mobile devices

---

### 🎨 Custom Style Not Loading

**Symptoms:** Style preset shows "default" despite config changes.

**Solutions:**
```bash
# Verify preset name matches exactly (case-sensitive)
CUSTOM_STYLE_PRESET = "kj_dark_v2"   # ✅ Correct
CUSTOM_STYLE_PRESET = "KJ_Dark_V2"   # ❌ Wrong — case mismatch

# Check the presets folder path
ls ./presets/
# Should output: default.json  kj_dark_v2.json  admin_gold.json
```

---

### 📱 Mobile Overlay Not Appearing

**Symptoms:** Executor says "Injected" but no floating panel in-game.

**Solutions:**
- Re-grant overlay permission: **Settings → Apps → [Executor] → Draw Over Other Apps → Allow**
- Reboot the device and re-inject
- Ensure `ADMIN_PANEL_AUTOLOAD = true` in your config

---

### 🔑 KJ Style Showing as Locked

**Symptoms:** KJ style appears greyed out in the Styles tab.

**Solutions:**
- Set `KJ_STYLE_ENABLED = true` in your config and re-inject
- Ensure you are running v3.4.0 or later
- The feature requires the full character data to load — wait until you are in an active session, not the loading screen
